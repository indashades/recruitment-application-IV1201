# Project

## Run locally

### Server
```
cd server
npm install
npm run db:init / npm run db:reset
npm run dev
```

### Client
```
cd client
npm install
npm start
```
## Cloud

A deployed API is available at: https://recruitment-application-iv1201.onrender.com <br>A deployed Client app is available at: https://recruitment-application-iv1201-client.onrender.com/

Note 1: The server may take a few minutes to start after inactivity, avoid spamming requests. <br>Note 2: The cloud server and client may need manual updating after commits.
