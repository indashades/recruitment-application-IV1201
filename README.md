# Project

## Run locally

### Server
```
cd server
npm install
npm run db:init / npm run db:reset
npm run dev
```

### Client
```
cd client
npm install
npm start
```
## Cloud server

A deployed API is available at: https://iv1201-api.onrender.com

Note: The server may take a few minutes to start after inactivity, avoid spamming requests.
